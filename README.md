# React Calculator

A modern calculator built using React.js and Vite.

## Features

- Basic arithmetic operations
- Dark/Light theme
- Calculation history
- Copy result
- Responsive design

## Tech Stack

- React.js
- Vite
- CSS3
- JavaScript
